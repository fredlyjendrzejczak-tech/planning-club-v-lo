# Planning Club Velo

Application partagee pour consulter le planning hebdomadaire du club et enregistrer le detail des seances.

## Envoyer sur GitHub

1. Ouvrir le dossier `A_ENVOYER_SUR_GITHUB_PLANNING_CLUB_VELO`.
2. Selectionner tout le contenu du dossier.
3. Le deposer dans la page GitHub avec `uploading an existing file`.
4. Cliquer sur `Commit changes`.

## Heberger avec Render

1. Aller sur Render.
2. Creer un `New Web Service`.
3. Connecter le depot GitHub.
4. Render detectera automatiquement le fichier `render.yaml`.
5. Lancer le deploiement.

Les seances sont enregistrees dans un petit fichier JSON cote serveur.
